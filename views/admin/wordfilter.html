Word Filter

Substitutions are applied to post bodies before markup rendering. Matching is case-insensitive and whole-word only (word boundaries). The replacement string is displayed as-is — you can use this for humorous substitutions (classic 4chan style) or to scrub slurs. Inactive rules are stored but not applied.

Examples: trudeau → the drama teacher    poilievre → Poily
