Verified Politicians

Each verified politician is issued a non-transferable Soulbound Token on-chain. The token holds their display label (shown on all their posts) and an expiry date matching their term end. You can burn any token instantly if they are ousted or resign. All transactions are signed by your MetaMask wallet — the server never holds a private key.
